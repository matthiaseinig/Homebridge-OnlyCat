# Examples

## `homebridge-config.json`

A minimal `config.json` showing how the OnlyCat platform plugs into Homebridge. Use it as a starting point if you edit `config.json` directly.

If you use the **Homebridge UI** (the standard `homebridge-config-ui-x` web interface), you don't need this file — the UI will write the platform block for you when you configure the plugin through the Plugins tab.

> ⚠️ Replace `PASTE_YOUR_ONLYCAT_TOKEN_HERE` with your real token before saving. Treat the token as a password.
